# MandelViewer

![mandelbrot set](https://upload.wikimedia.org/wikipedia/commons/thumb/2/21/Mandel_zoom_00_mandelbrot_set.jpg/300px-Mandel_zoom_00_mandelbrot_set.jpg)

## requirements
- Python 3.10
- pytest `pip install -U pytest`

## getting started

run the app using
```
python3 src/mandelview.py
```

run the tests using
```
pytest test
```
